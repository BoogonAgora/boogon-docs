## 创建一个Vue应用

我们按照[官方](https://cn.vuejs.org/)提供的方法进行创建。

> 前提条件：熟悉命令行，以及安装合适版本的Node.js运行时。


{{< tabs items="NPM,PNPM,YARN" >}}

  {{< tab >}}```npm create vue@latest```{{< /tab >}}
  {{< tab >}}```pnpm create vue@latest```{{< /tab >}}
  {{< tab >}}```yarn create vue@latest```{{< /tab >}}

{{< /tabs >}}

这一指令会安装并执行`create-vue`官方项目脚手架工具。执行时，你将看到一些可选功能提示信息。
